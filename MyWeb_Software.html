Visual Studio Code

一つ目は「Visual Studio Code」！
こちら僕の学科ではHTMLの授業でダウンロードしましたが、環境設定をすれば、いろんな言語を使えるという優れもの。
沢山の人に使われているので、ソフト内でこまったことがあったり、変えたい機能などを検索すると、だいたいヒットするのでとても便利なソフトです！
僕はJavaの授業、このソフトで受けてます！

Visual Studio

二つ目は「Visual Studio」！
こちらのソフト、Codeとめちゃくちゃロゴが似てますが、全くの別物。
Codeの方はプログラマ向けのテキストエディタで、付かない方は統合開発環境。
Windowsで動きますが、Windowsアプリケーションだけでなく、WebアプリケーションやLinux、Android、iOS、一部のmacOS向けのアプリも作成できます!
僕は主にC++で使っていますが、セミコロン(;)を入力してEnter押すだけできれいにインデントしてくれるのできれいにコーディングできるところが僕は好きです！

Dream Weaver

Adobe Dreamweaverを皆さんご存じでしょうか？
僕は知りませんでした！
写真のお仕事をするときにAdobeのソフトをよく使うのですが、まさか、AdobeでWeb作成ができるなんて。使うしかないやろ。で、ダウンロードしました。
このソフト何がすごいかといいますと、
編集と同時にプレビューが表示されるのです！
え？「そんなのVS Codeにも拡張機能であるやん？」だって？？？チッチッチ
VS Codeでは編集した内容をプレビューに表示するにはいちいち「保存」をしていましたよね？
しかしDreamweaverでは保存しなくてもリアルタイムでプレビューも変化していくのです！
しかも超多機能！
